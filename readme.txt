   Genera automáticamente meta descripciones optimizadas para SEO utilizando IA.

   == Description ==
   SEO Meta Description Generator es un plugin que utiliza inteligencia artificial para crear meta descripciones optimizadas para SEO para tus posts y páginas.

   Características:
   * Generación automática de meta descripciones
   * Integración con OpenAI para descripciones mejoradas por IA
   * Opción de generación masiva para contenido existente
   * Personalización de la longitud de las descripciones

   == Installation ==
   1. Sube los archivos del plugin al directorio `/wp-content/plugins/seo-meta-description-generator`
   2. Activa el plugin a través del menú 'Plugins' en WordPress
   3. Ve a 'Ajustes' > 'SEO Meta Desc' para configurar el plugin

   == Frequently Asked Questions ==
   = ¿Necesito una clave API de OpenAI? =
   Sí, para utilizar la función de generación con IA, necesitarás una clave API de OpenAI.
